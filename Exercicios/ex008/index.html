trabalhando com listas

listas ordenadas

para abrir a lista, se usa a tag <ol> e para listar os itens é a tag <li> ↓

  1. acordar
  2. escovar os dentes
  3. ligar o note
  4. estudar
  5. almoçar

o parâmetro type muda os indicadores numéricos da lista, exemplo: algarismos romanos, números e etc.

Já o parâmetro start indica a partir de qual número ou algarismo a lista começa

Listas não ordenadas

para listas não ordenadas é a tag <ul>

  • re:zero
  • code geass
  • Fullmetal alchemist brotherhood

outras situações

para fazer uma sublista dentro desta, basta usar <ol> denovo, e usar <li> também

  1. lista
    • isso é uma sublista
